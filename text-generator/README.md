## Getting Started

### Prerequisites

- npm
  ```sh
  npm install npm@latest -g
  ```

### Installation

1. Clone the repo
   ```sh
   git clone https://github.com/ozanakayaci/redux-projeler.git
   ```
2. Project directory
   ```js
   cd redux-projeler/text-generator
   ```
3. Install NPM packages
   ```sh
   npm install
   ```
4. Runs the app in the development mode.

   ```sh
   npm start
   ```
